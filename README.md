# Guru Ramu — Portfolio

Personal portfolio site showcasing AI agent infrastructure and Web3 engineering work.

Built with Next.js 15, HeroUI, Tailwind CSS, and Framer Motion.

## Development

```bash
pnpm install
pnpm dev        # runs on http://localhost:3000
pnpm build      # production build
```

## Content
All portfolio data (experience, projects, skills, contact) lives in [`data/index.ts`](data/index.ts).
